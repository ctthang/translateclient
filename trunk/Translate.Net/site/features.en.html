Features

Translate.Net support next translation services:
  • translate.google.com  - translator. 28 language pairs. English to 13 languages and back + 4 cross languages. Supported only translator, dictionary support planned. Copyright © 2008, Google
  • translate.meta.ua - translator. 20 language pairs. Ukrainian to 4 languages and back, Russian to 4 language pairs and back + 4 cross languages. 21 subjects. Copyright © 1998-2008, <META>
  • pere.org.ua  - translator. English, Ukrainian, Russian cross translations with socio-political subject. Copyleft (L) 2003-2007, Kyiv, Valentyn Solomko
  • r2u.org.ua  - bilingual dictionary. Russian to Ukrainian bilingual dictionaries printed at 1924-1930. Detail in ukrainian.   Also search in ukrainian words. Development - Andriy Rysin, KIS
  • www.slovnyk.net  - great ukrainian monolingual dictionary. 207000+ words. Copyright © 2005, VTF Perun, © 2006-2008, Oleg Dmitriev
  • slovnyk.org  - bilingual dictionary. 32 languages. 992 languages pairs. Copyleft (L) 2000-2008, Kyiv, Valentyn Solomko
  • dict.linux.org.ua  - bilingual dictionary. English-Ukrainian technical dictionary. Development - Andriy Rysin, www.linux.org.ua community, used dictionaries. Also search in ukrainian words.
  • rosukrdic.iatp.org.ua  - bilingual dictionary. Russian-Ukrainian dictionary of constant expressions. 6208 articles. © 2006-2008, Anatolyi Evpak
  • more services is planned
The user interface is intuitive and quickly learnable without assistance. Supported English and Ukrainian GUI.



User interface enhanced with many smart improvements:
  • history of used language pairs which sorted by last use time
  • input locale catching - when user change input locale, the latest language pair from history is selected. When user select language pairs - input locale changed accordingly
  • system hotkeys (Ctrl+C+C and\or Ctrl+Ins+Ins) allow to activate Translate.Net and perform translation from any program easy
