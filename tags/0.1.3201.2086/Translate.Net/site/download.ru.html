Скачать
 

Скачать архив для запуска с флеш-диска

Распакуйте архив в какой-нибудь каталог и запустите X-Translate.Net.exe. Все настройки сохраняются в подкаталоге, поэтому программа может быть запущена на компьютере, соответствующем минимальным требованиям, без установки. Данная версия была изначально создана Danilo Leggieri, как часть большой коллекции портативных программ - winPenPack
 




Исходный код доступен на сайте http://code.google.com/p/translateclient/ .
Для компиляции необходима Visual Studio 2005 или .Net SDK (доступно бесплатно от Microsoft) и бесплатная интегрированная среда разработки SharpDevelop . Инсталлятор изготовлен с помощью конструктора инсталляций Inno Setup.
