About

Translate.Net is free and open source client to many translating services available online (translators, monolingual and bilingual dictionaries).
Already supported 17 services, 35 languages, 1352 translations directions.
It designed with smart user interface to allow use different services from single place.

    • In comparison with locally installed dictionaries and translators Translate.Net show reduced requirements for hard drive memory. Also ability to retrieve results from different sources improve reliability of translation. On other hand Translate.Net require working connection to internet for processing of queries and retrieving translating results.  So if you can't be online when needed or require improved safety of yours translations please use other solutions.

    Translate.Net may show ads based on context of your queries, languages and subjects you select. Translate.Net don't collect any personal data and don't use any aggressive advertisement technique. Ads shown in result view with look and behavior as any advertisement at web sites.

    Tool developed in C# language with SharpDevelop IDE on Microsoft's .NET platform, installer created with Inno Setup installation builder, for image processing used Paint.Net image editor. For site design used KompoZer html editor.
