If you installed Translate.Net not from home site, please press menu "Help\Check updates" for installing latest version.

Small tip : some extra languages (like "English_US", "English_GB", "Chinese_CN", "Chinese_TW") supported by small count of engines, if you not satisfied with result, please check with common names, i.e. "English", "Chinese".
