Thank you for installing Translate.Net version 0.1.3201.2086 (released at 2008-10-06).

This is free open source project with goal to simplify access to many language resources available online.

Additional information about Translate.Net:

  • Changelog - description about new features and changes in this version.
  • Online documentation about use of Translate.Net
  • Translate.Net also has portable version. You can download it here.

  • Create localization for Translate.Net in your language. I am will be pleased to have localization to many languages as possible. Documentation for localization stored at Localization Tutorial
  • Suggest online translation engine or dictionary you like

Translate.Net is free open source project and I encourage everyone to donate Translate.Net project.
